Lab 6: Interactive Mapping

Introduction

This lab combines interactive web mapping with custom cartographic design. I created a personalized basemap, built a dataset of meaningful locations from my hometown, and integrated both into an interactive web map. The lab consists of three main parts: custom basemap design, hometown locations dataset, and Python-generated interactive map.

Part 3: Python Interactive Map

Technical Implementation

I used Python with the following libraries to create the interactive map:

  • Pandas: For reading and managing the CSV dataset
  • Requests: For accessing the Mapbox Geocoding API
  • Folium: For creating the interactive web map

Key Features

  • Geocoded all addresses using the Mapbox Geocoding API
  • Integrated my custom Mapbox basemap as the base layer
  • Color-coded markers by location type
  • Created interactive pop-ups with images and personal descriptions
  • Exported the final map as an HTML file

Code Explanation

The Python script follows this workflow:

  1. Read the CSV file using pandas
  2. For each location, send the address to Mapbox Geocoding API to get lat/long coordinates
  3. Create a Folium map centered on my hometown with my custom Mapbox basemap
  4. Add markers with different colors based on location type
  5. Create HTML pop-ups with location name, description, and image
  6. Save the map as an HTML file

The complete Python script is available in hometown_map.py.
